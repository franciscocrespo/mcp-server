# mcp-server

Servidor MCP implementado con NestJS

## Instalación

```bash
npm install
```

## Ejecución

```bash
# desarrollo
npm run start

# modo watch
npm run start:dev

# producción
npm run start:prod
```

## Tests

```bash
# tests unitarios
npm run test

# tests e2e
npm run test:e2e

# cobertura de tests
npm run test:cov
```
